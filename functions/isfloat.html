ISFLOAT¶

An ISFLOAT function can be used to check whether a number is a float or int. The function is commonly used as a part of a conditional statement within an IF.

Usage¶

ISFLOAT(col) : Boolean

Examples¶

In the example below, we take an integer (‘foo’), change it using a float type conversion into a float (at which point it becomes 5.0) and then use the ISFLOAT within an IF statement to evaluate if it is a float or not. (It is.)

gorrow 1,1,1 | CALC foo 5 | CALC bar float(foo) | CALC float? IF(ISFLOAT(bar),'true','false')
